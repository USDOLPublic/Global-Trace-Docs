5. Assessment configuration

During Role Definition, the super administrator configures if a role will require completion of an assessment or self assessment questionnaire when the user is onboarded. Each role that was provided permission to undertake an assessment will appear in the table.

In this step we use an offline excel document to define an assessment for each role. Templates can be downloaded using the “Manage attachments” button in the table.

Assessment Definition Template After downloading the template you will see an excel document with two tabs: Labor and Product. All questions provided in the labor tab will appear under the heading “Labor” during the SAQ flow (in user onboarding), and all questions provided in the product tab will appear under the heading “Product”.

Column Title Description
A Index Question number
B Question
C Method of collecting data How do we collect data? Platform or In-person
D Conditional question Either ‘Y’ or ‘N,’ If ‘Y,’ then we must mention question number in Go to (Column H) for routing
E Response Type Single select, Multi select, Boolean, Integer
F Response options Responses to choose
G Option Type Input as “Other” if we need to get input from user
H Go to Determines routing based on responses to the question
I Risk category (Optional) Low/Medium/High
J Indicator (Optional) From the Indicator List
K Sub-indicator (Optional) From the Indicator List

Columns I, J, and K are used to signal that a question should contribute to the overall risk for the supplier and are optional. Note that if a risk, indicator or sub-indicator is provided, then the expectation is that all three items will be completed.

Once an SAQ has been successfully uploaded, an Upload SAQ translation link is enabled on the super administrator flow.

Facility Group Templates

For each role that is configured for the administrator to upload the assessment, an additional template is required on top of the excel described above. The assessment definition template is used to describe the questions that will be asked. The facility group template is a formatted template that is used to collect data from users, and then is uploaded by the administrator after completion.
